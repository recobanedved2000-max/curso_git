comandos git -> teste 

ls - listar arquivos, pasta atual = ls -a arquivos ocultos
cd - muda de diretorio
pwd - caminho das pastas até a pasta atual 
ctrl + l - LIMPA TELA TERMINAL
mkdir - cria pasta = mkdir "nome da pasta"
cat - ler arquivo 
git init . - cria uma pasta repositorio especial github
rm - remove arquivo 
git status - estado atual da pasta git
git add "nome arquivo" - inclue arquivo para commit 
GIT LOG - 	
git diff - mostra o que esta adicionado no arquivo atual, porem não commitado
git reset - retira o arquivo de pronto para commit 
git commit -m "nome da ação no commit"
git checkout - b 'nome da ramificação' - brench separada e/ou secundaria
git merge - junta os branchs(trabalhos) 
git branch -d 'nome branch'- exclui a branch
